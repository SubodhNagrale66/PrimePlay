PRIME PLAY - A VIDEO STREAMING WEBSITE

NOTE- Internet service required since images and movies are loaded via url links.

Starting Page-
"index.html" used for Login.

Folders:
html- to store html files
css - to store css files
images - to store images 
js - to store external JAVSCRIPT FILES

Note: Cloudinary.com is used to store most of the images so that size of the file will be under 10MB

Navigation-
index.html(Login) -> signup.html(New User Signp) -> plans.html (To select plans subscription) -> payment1,payment2,payment3,payment4.html(For Payment Section) -> Successful.html(represents successful payment) ->**redirected to index.html(for login) -> home.html(On Successfull Login using credentials provided on Signup.html) -> movie/*.html(on clicking image of movie/series on home page) -> Signout button( In Hamburger menu for log out)
